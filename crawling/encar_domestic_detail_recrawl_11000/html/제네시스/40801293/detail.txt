현재 고객님께서 보시고있는 차량은 100% 실매물임을 이름을 걸고 약속드립니다. 

▶차량설명 
-사고여부 : 무사고 차량입니다. 
-차 종 : 제네시스 G80 3.3 GDI 럭셔리 
-연 식 : 2016년 9월 (17년형) 
-색 상 : 인기있는 은색입니다. 
-주행거리 : 101,167km 로 짧습니다. 
-내 / 외관 : 실내와 실외가 깨끗합니다.
-관리상태 : 모든 정비를 마친 상태입니다.

▶옵션사항 
-외관/내장사양 : 헤드램프(HID), 전동접이 사이드 미러, 알루미늄 휠, 열선 스티어링 휠, 전동 조절 스티어링 휠, 패들 시프트, 스티어링 휠 리모컨, ECM 룸미러, 하이패스, 파워 도어록, 파워 스티어링 휠, 파워 윈도우 
-안전사양 : 에어백(운전석, 동승석), 에어백(사이드), 에어백(커튼), 브레이크 잠김 방지(ABS), 미끄럼 방지(TCS), 차체자세 제어장치(ESC), 타이어 공기압센서(TPMS), 주차감지센서(전방, 후방), 후방 카메라
-편의/멀티미디어사양 : 크루즈 컨트롤(일반), 전자식 주차브레이크(EPB), 자동 에어컨, 스마트키, 무선도어 잠금장치, 레인센서, 오토 라이트, 내비게이션, 앞좌석 AV 모니터, 블루투스, CD 플레이어, USB 단자, AUX 단자 
-시트사양 : 가죽시트, 전동시트(운전석, 동승석), 열선시트(앞좌석, 뒷좌석), 메모리 시트(운전석), 통풍시트(운전석, 동승석)

▶차량의 특징 
-역동적인 바디라인을 강조한 디자인입니다. 
-대형차에서 볼수 없던 편의사양 
-스포티한 주행이 가능 

▶차주정보 
-운행자정보 : 1인 소유
-운행용도 : 출퇴근용으로 사용하여 주행거리가 짧습니다. 
-구입방법 : 신차로 구매하여 운행하던 차량입니다. 
-판매이유 : 이번에 중고차를 구매하면서 판매하게 되었습니다. 

▶문의방법 
-연락주시면 친절하고 상세하게 설명을 드리도록 하겠습니다. 전화가 부재중일시 문자를 남겨주시면 확인후 전화드리겠습니다. 
-차량은 대구 달서구에서 보실 수 있어요. 
-가격은 절충이 가능합니다. 
-차량 시승도 가능하며 정비업소에서 차량상태 확인도 가능합니다. 
-현재 타고 계신 차량과 교환도 가능하며, 대출을 통한 차량구매도 가능합니다.
